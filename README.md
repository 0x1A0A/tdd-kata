# TDD kata

This repo contain solution of problem from checkio, leetcode and codeforces. Which I using to practices Test Driven Development (TDD).

## Language
- C,C++
- Ecmascript (typescript)
- Rust

## How to run

### C,C++
Dependency
- cmake
- make or other build system

```bash
cd c
cmake -S . -B build
cd build
make && ./src/tdd
```

### Ecmascript (typescript)
Dependency
- Deno

```bash
cd esm
deno test
```

### Rust
Dependency
- rustup

```bash
cd rust
cargo test
```
